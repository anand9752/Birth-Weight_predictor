📌 What each input means
  • Bwt (Birthweight): Birth weight of the child, measured in ounces (1 ounce ≈ 28 grams).
  • Gestation: Length of pregnancy, measured in days (from conception to birth).
  • Parity: Binary indicator for first pregnancy (0 = first pregnancy, 1 = not first).
  • Age: Mother's age in years.
  • Height: Mother's height in inches.
  • Weight: Mother's weight in pounds.
  • Smoke: Binary indicator for whether the mother smokes (0 = No, 1 = Yes).
📌 Model Used

Linear Regression model is used to predict birth weight based on input features.
