About This Project
The Birth Weight Predictor is a data science project that demonstrates how machine learning can be used to make health-related predictions. It is built with Flask, Python, and a Linear Regression model trained on real birth data.
Purpose
This project is for educational and demonstration purposes, showing the process of data preprocessing, model training, and web deployment.
Features
  • Clean, modern, and responsive web interface
  • Detailed input validation and tooltips
  • Instant prediction of birth weight in ounces
  • Project and model documentation
