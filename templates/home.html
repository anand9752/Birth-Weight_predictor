Welcome to the Birth Weight Predictor
This project leverages machine learning to predict the birth weight of a child based on maternal and pregnancy-related factors. It is designed for educational and demonstration purposes, showing how data science can be applied to real-world health data.
How does it work?
  • Enter details about the pregnancy and mother on the Prediction page.
  • The model uses a Linear Regression algorithm trained on real birth data.
  • Get an instant prediction of the expected birth weight (in ounces).
What inputs are required?
  • Gestation: Length of pregnancy in days (200-320)
  • Parity: Is this the first pregnancy? (0 = Yes, 1 = No)
  • Age: Mother's age in years (15-50)
  • Height: Mother's height in inches (50-80)
  • Weight: Mother's weight in pounds (90-300)
  • Smoke: Does the mother smoke? (0 = No, 1 = Yes)
